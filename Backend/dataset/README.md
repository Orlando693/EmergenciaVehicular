# Dataset de ejemplo

Esta carpeta contiene datos de entrenamiento de ejemplo para diagnosticos de incidentes vehiculares.

No esta conectada al backend, a Gemini, a la base de datos ni a ningun proceso automatico. Solo queda como material de referencia para una futura etapa de entrenamiento, pruebas o documentacion.

Archivos:

- `diagnosticos_entrenamiento.csv`: casos tabulares con sintomas, categoria y recomendacion.
- `incidentes_ejemplo.jsonl`: mismos casos en formato JSON Lines, util para pruebas de procesamiento.
